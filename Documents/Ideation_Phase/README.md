# Ideation Phase - Garage Management System (GMS)

This folder contains the ideation and conceptualization documents for the Garage Management System Salesforce project.

## Contents
- Brainstorming - Idea Generation.pdf: Identifies core problems, opportunities, and proposed solutions.
- Define Problem Statement.pdf: Defines the project goal, scope, and expected outcomes.
- Empathy Map Canvas.pdf: Highlights user perspectives and needs for better system design.

### Summary
The Ideation Phase establishes the foundation for the GMS by identifying the real-world challenges of garages and conceptualizing
an efficient Salesforce-based digital management solution.
